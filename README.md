# Htantabin Project

![Htantabin Logo](public/images/favicon/mstile-150x150.png)

## Overview

Htantabin is a voucher system for use at the Monasteries. It includes two types of vouchers.

## Features

- Lucky Draw voucher
- Pathan voucher
- User role based
- Print with PDF
